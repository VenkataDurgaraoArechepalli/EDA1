# EDA1
Exploratory Data Analysis (EDA) is the process of examining, cleaning, summarizing, and visualizing a dataset to understand its structure, identify patterns, detect missing values or outliers, and gain meaningful insights before building any machine learning model. 
EDA helps data analysts and data scientists understand the characteristics of the data and make informed decisions about data preprocessing and model selection.
In this project, Exploratory Data Analysis is performed on the Titanic dataset to analyze passenger information such as age, gender, passenger class, fare, and survival status.
The dataset is first explored using statistical summaries and data inspection techniques. Missing values are identified and handled appropriately to improve data quality. 
Various visualizations, including count plots, histograms, bar charts, box plots, and correlation heatmaps, are created using Matplotlib and Seaborn to reveal relationships between different variables.
The analysis provides valuable insights into how factors such as gender, passenger class, age, and ticket fare influenced the survival of passengers on the Titanic. These findings help in understanding the dataset and preparing it for further predictive modeling.
EDA is an essential step in the data science workflow because it improves data quality, uncovers hidden trends, and supports better decision-making during model development.
